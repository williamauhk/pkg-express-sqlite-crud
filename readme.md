# PKG Express

1. Support Build Express to Protable EXE file with PKG
2. Support CRUD with Sqlite
3. Support open brower after run EXE

# Installation
1. Clone
2. npm i

# Build 
1. nvm use 18
2. npm run build
3. you can see exe file on build folder
